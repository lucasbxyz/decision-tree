# Decision Trees & Random Forests
[![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/lucasbxyz/decision-tree/HEAD)

Predicting whether borrowers from LendingClub.com will fully repay their loans using Decision Trees and Random Forests. The dataset contains loan data from 2007–2010 with features like FICO score, interest rate, and loan purpose.
